# flask-blog
>파이썬 Flask 프레임워크를 이용한 방명록 웹사이트

http://ec2-python.for-dev.xyz
## Made with...
Backend Framework: **Flask**   
Database: **Sqlite3**   
ORM: **Sqlalchemy**   
Frontend: **Jinja2 + Bootstrap4**
## Hosted with...
AWS EC2 Instance   
Nginx (as a Reverse Proxy)
uWSGI (as a MiddleWare)
